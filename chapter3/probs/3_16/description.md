### 问题描述

假设我们有基于数组的表arr[0, n-1]，并且我们想要*删除所有的相同的元素*。
`lastPosition`的初始值是n-1，随着元素的删除将变得越来越小。
考虑如下的代码片段
```c
    for (i = 0; i < lastPosition; i++) {
        j = i + 1;
        while (j < lastPosition) {
            if (arr[i] == arr[j]) {
                deleteEle(j);
            }else {
                j++;
            }
        }
    }
```
其中`deleteEle`函数作用是*删除索引上的元素，并且将表缩小*。
